Concept 3 · Bold travel-game energy

Jetstream

The world is the league table. Every country is points on the board — now go get them.

The vibe: sports-graphics broadcast meets boarding pass meets achievement screen. Big chunky Nunito numerals, sticker badges, XP bars with milestone ticks, and a match-day fixture for territory battles. It treats travel the way a 15-year-old already thinks about it — a game with a scoreboard — without ever tipping into childish.

How it serves the principles: transparency becomes the star of the show — Japan's score is a friendly Q&A that ends in one line of visible arithmetic, like a referee showing their working. Fairness reads at a glance: caps, floors and multipliers appear as plain ink chips, never hidden. And every country matters because every country is bait — hover any of the 195 and the map tells you exactly what it's worth, from Spain's 16.0 to Mongolia's 186.6.

What changes vs today's plain Tailwind UI: grey utility tables become white cards on warm paper with one hero number each; the country header becomes a boarding pass with a tear-off score stub; progress becomes XP bars; and the battle page becomes a broadcast fixture — possession bar, VS lockup, overtime chips. Same data, ten times the pulse.

What to notice

  • Adventure bait: hovering any country tells you what it's worth — Mongolia is worth up to 186.6 pts — go get it. The map sells the next trip.
  • One hero number: Total Points gets the gradient band and an XP bar with milestone ticks — 37.4 pts to the 750 Club turns the score into a chase.

  • Sticker badges celebrate milestones in amber only — data colours stay reserved for the map, so nothing lies.
  • Legend chips are rounded stickers; visited teal vs runway grey is readable at arm's length.

What to notice

  • The boarding pass header — perforated stub tears off the number that matters: 85.2 banked of 137.8 max, with milestone ticks on the XP bar.
  • The maths is shown, not hidden: the breakdown ends in ×4.4 × (6.94 + 5.16) = 53.2 on a highlighted strip — a 15-year-old can check the referee's working.
  • Q&A voice: each factor is a question a person would actually ask, answered in one sentence, valued in one ink chip.
  • 36% explored rides the exploration bar as a pill — Japan still owes charlie 52.6 points of bonus, and the page makes that feel like unclaimed loot.
  • Prefectures are individually priced (Tokyo 9.4 pts, Fukuoka 3.4) so the next trip inside Japan has a scoreboard too.

What to notice

  • Match-day fixture: avatar circles, a big VS lockup, and scores in 900-weight tabular numerals — it reads like a football broadcast graphic.
  • The possession bar splits 51.0 / 49.0 with a halfway-line marker — “You're ahead by 27.5!” sits on the winning side.
  • Overtime chips make contested countries feel like extra time, and the rule is spelled out in one sentence.
  • The reassurance line — battles never change anyone's Travel Points — keeps it fun-but-fair, right under the title.
  • Battle colours are the app's existing validated set (blue / red / violet / runway grey) with all labels in ink, never in series colour.

Implementation notes

Everything above maps onto the existing React + Tailwind v4 app with a token swap and three component restyles.

Tailwind v4 @theme tokens
@theme {
  --color-paper:     #fdfbf7;  /* page background          */
  --color-ink:       #1f2430;  /* headings, values         */
  --color-mute:      #5b6472;  /* secondary text           */
  --color-coral:     #ff6b4a;  /* brand gradient start     */
  --color-violet:    #7c5cff;  /* brand gradient end       */
  --color-jet:       #0f9d8f;  /* map: visited             */
  --color-runway:    #e7e4dc;  /* map: unvisited/unclaimed */
  --color-you:       #3b82f6;  /* battle: you              */
  --color-rival:     #ef4444;  /* battle: Maya             */
  --color-contested: #a855f7;  /* battle: contested        */
  --color-badge:     #f59e0b;  /* badges & streaks only    */
  --font-display: "Nunito", system-ui, sans-serif;
  --radius-card: 20px;
  --shadow-card: 0 2px 8px rgb(31 36 48 / 0.08);
}
